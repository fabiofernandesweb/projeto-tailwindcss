# projeto-tailwindcss
Projeto com Tailwind CSS focado em responsividade e UI moderna.

---

### Tecnologias utilizadas
- HTML5
- Tailwind CSS
- JavaScript (opcional)

---

### Preview
- https://github.com/fabiofernandesweb/projeto-tailwindcss/

---

### Objetivo

Este projeto foi criado para:
- Aprender e praticar Tailwind CSS
- Desenvolver layouts responsivos
- Melhorar organização e reutilização de estilos
- Simular um projeto real para portfólio

---

### Responsividade

O projeto foi desenvolvido utilizando o conceito mobile-first, garantindo boa experiência em:

- 📱Celulares
- 💻Tablets
- 🖥️Desktops

---

### Funcionalidades
- Menu responsivo (hambúrguer)
- Layout moderno
- Estilização com Tailwind
- Hover effects
- Componentes reutilizáveis
